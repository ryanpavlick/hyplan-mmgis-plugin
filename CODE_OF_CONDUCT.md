# Code of Conduct

hyplan-mmgis-plugin aims to be a respectful, collaborative project for
scientific and software work.

## Our expectations

- Be respectful and professional in technical discussion.
- Assume good intent and give constructive feedback.
- Welcome questions from contributors with different levels of
  experience.
- Focus critique on ideas, code, documentation, and behavior, not on people.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks
- Trolling or sustained disruptive behavior
- Demeaning comments about experience level, identity, or background
- Publishing private information without permission

## Scope

This code of conduct applies to project discussions, issues, pull
requests, reviews, and other hyplan-mmgis-plugin community spaces.

## Reporting

If you experience or observe unacceptable behavior, contact the project
maintainer through the repository contact channels or by opening a
private report if such a channel is available.
